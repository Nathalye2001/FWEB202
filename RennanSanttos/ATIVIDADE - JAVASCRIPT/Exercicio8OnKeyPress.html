EXEMPLO ON KEY PRESS:

O evento onkeypress ocorre quando o usuário pressiona uma tecla (no teclado). Dica: a ordem dos eventos relacionados ao evento onkeypress: > onkeydown > onkeypress > onkeyup Nota: O evento onkeypress não é disparado para todas as teclas (por exemplo, ALT, CTRL, SHIFT, ESC) em todos os navegadores. Para detectar apenas se o usuário pressionou uma tecla, use o evento onkeydown , porque ele funciona para todas as teclas.

A função é disparada quyando o usuário pressiona uma tecla no campo de entrada.
