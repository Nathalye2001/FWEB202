Projeto de Conclusão da Matéria de Programação Web

Essa tarefa tem como finalidade mostrar os conhecimentos adquiridos, a fim de avalidar as técnicas dos alunos na utilização dos recursos de HTML,CSS e JS.

Cadastros

Na opção Cadastro você ira encontrar dados ficticios sobre personas conhecidas e desconhecidas dados esses que não são confirmados e podem não ter ligação com a vida real.

Fale Conosco

Essa opção permite simular uma avaliação do site ou até mesmo sugestão de melhoria.Os dados seram enviados aos desenvolvedores para planejar novas atualizações levando em consideração todos os dados coletados pelo site.

Séries

Nesta página você ira encontrar uma pequena sinopse de algumas séries de diversos gêneros, além de ser apresentado alguns imagens das mesmas.A página tem como ideia central fazer sugestões para os interressados pelo universo das plataformas voltadas a exibição desse tipo de conteudo.

Cabelos Cacheados

Está em aparticular é uma página que só terá utilidade para pessoas que tem o cabelo cacheado.Nela será apresentado um tutorial de finalização de cabelo além de conter um link que levará os interessados a um vídeo especifico sobre o assunto.
